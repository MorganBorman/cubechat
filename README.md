cubechat
====================

A chat client for all (eventual goal) cube 2 engine games.

###Currently supported cube 2 games

* Sauerbraten Justice
* Sauerbraten Collect

* Red Eclipse Elara

* Assaultcube V1.1.0.4

###Getting started

To run:
```bash
    cd src
    python cubechat.py <hostname> <port> [name]
```
